## 🚦 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/ParkingGarage.git
   cd ParkingGarage
   ```

2. **Install CCPM**
   ```bash
   curl -sSL https://raw.githubusercontent.com/automazeio/ccpm/main/install.sh | bash
   ```

3. **Initialize project management**
   ```bash
   /pm:init
   ```

4. **Start development**
   ```bash
   npm install
   npm run dev
   ```
